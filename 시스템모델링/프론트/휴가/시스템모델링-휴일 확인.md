시스템 명칭: 휴가관리시스템-미니
모델 명칭:  휴일 확인
레벨: 시스템-프론트, 백엔드

# 개요 및 목표
1. 시스템은 특정 날짜의 휴일 여부를 반환한다.

# 선행 조건

# 시작 이벤트
1. 시스템이 모델을 호출한다.
	1. {파라미터} thisDay : 날짜

# 모델 시나리오
1. 시스템은 isHoliday (휴일 여부)를 false로 설정한다.
2. 법정공휴일 + 사용자 설정 공휴일이다.
	1. {반환} isHoliday : true
3. 주말이다.
	1. {반환} isHoliday : true
4. {반환} isHoliday
